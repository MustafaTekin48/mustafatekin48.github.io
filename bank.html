💳 Banka Uygulaması (Kumbara)

📌 Açıklama

Banka Uygulaması (Kumbara), kullanıcıların para biriktirmesini, harcamalarını yönetmesini ve finansal hedeflerine ulaşmasını kolaylaştırmak amacıyla geliştirilmiş bir kişisel finans uygulamasıdır.

🛠️ Kullanılan Teknolojiler

  • React Native – Mobil uygulama geliştirme
  • JavaScript / TypeScript
  • Firebase – Veritabanı ve kimlik doğrulama
  • Expo CLI – Geliştirme ortamı

🚀 Uygulama Özellikleri

  • ➕ Para ekleme ve ➖ Para çekme işlemleri
  • 🎯 Hedef belirleme (örneğin: Kasım’a kadar 1000₺ biriktir)
  • 📈 Hedef ilerlemesini görselleştirme
  • 📊 Geçmiş işlemleri listeleme
  • 🔔 Hatırlatıcılar (isteğe bağlı)
  • 🔐 Güvenli giriş ile kullanıcı koruması
