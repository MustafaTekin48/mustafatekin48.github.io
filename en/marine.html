🌊 Marine Field Team App

📌 Description

Marine Field Team App is a mobile application designed to help field teams in the maritime industry manage their tasks efficiently. It enhances operational productivity through features such as task assignment, GPS location reporting, offline/online data entry, and real-time notifications.

🛠️ Technologies Used

  • React Native – Cross-platform mobile development
  • JavaScript / TypeScript
  • Redux – State management
  • Expo CLI – Development environment

🚀 Features

  • ✅ Task assignment and status tracking
  • 📍 Real-time GPS location reporting
  • 📝 Data collection via forms
  • 📡 Offline/online support
  • 🔔 Instant notifications
  • 🔐 Secure login and authentication
