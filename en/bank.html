💳 Bank App (Kumbara)

📌 Description

Bank App (Kumbara) is a personal finance application designed to help users save money, manage expenses, and reach their financial goals with ease.

🛠️ Technologies Used

  • React Native – Mobile app development
  • JavaScript / TypeScript
  • Firebase – Database and authentication
  • Expo CLI – Development environment

🚀 App Features

  • ➕ Add money and ➖ Withdraw money operations
  • 🎯 Set saving goals (e.g., Save 1000₺ by November)
  • 📈 Visualize goal progress
  • 📊 List past transactions
  • 🔔 Optional reminders
  • 🔐 Secure login to protect users
