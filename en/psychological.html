🧠 Psychological Mobile App

📌 Description

Psychological Mobile App is a mobile application that uses image processing technology to analyze users' facial expressions and detect basic emotions. This helps users gain emotional awareness, track their mood, and support their psychological well-being.

🛠️ Technologies Used

  • Flutter – Cross-platform UI development
  • Dart – Main programming language
  • Android Studio – Development and testing environment
  • Image Processing – Real-time camera-based analysis

🚀 Features

  • 🎥 Emotion detection via facial expressions
  • 📈 Mood history tracking
  • 📊 Emotion analysis reports
  • 🔔 Personalized suggestions
